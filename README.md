## Contenido del respositorio: 
Aquí encontrarás los archivos de un proyecto grupal realizado por Micaela Lafratta Ramos y Mayka Ramos sobre el análisis y visualización del porqué del éxito de la serie Friends.

## Archivos:
Encontrarás: 
- Carpeta con datasets
- Carpeta con fotos
- Versiones anteriores trabajadas sobre Tableau Public
- Archivo definitivo dashboards Friends llamado "Friends_Exito serie y curiosidades".

Se puede encontrar la versión pública del tablero en [Tableau Public:] 
(https://public.tableau.com/app/profile/micaela.lafratta/viz/Friends_Exitoserieycuriosidades/CentralPerk?publish=yes)
